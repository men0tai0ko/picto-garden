## 1. ペット

### 1.1 データ構造

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string/number | 一意識別子 |
| type | enum(5種) | 種類 |
| level | number | ペットレベル |
| hp | number | 体力（上限100） |
| mp | number | スタミナ（上限100） |
| attack | number | 攻撃（上限100） |
| defense | number | 防御（上限100） |
| hunger | number | 空腹度 |
| personality | enum(5種) | 性格（ランダム決定） |
| rarity | string | レア度 |
| imageData | Blob | 元画像（IndexedDB保存） |

### 1.2 ペット種類（5種）

| 種類 | 輪郭特徴 | ゲームへの影響 |
|------|---------|--------------|
| ドラゴン系 | 縦横比縦長・エッジ密度中 | なし（図鑑のみ） |
| 鳥類系 | 重心上・縦横比横長 | なし（図鑑のみ） |
| 野獣系 | 重心中・エッジ密度高 | なし（図鑑のみ） |
| スライム系 | 縦横比正方形・エッジ密度低・重心下 | なし（図鑑のみ） |
| 精霊系 | エッジ密度最低・輪郭不明瞭 | なし（図鑑のみ） |

### 1.3 性格（5種）

| 性格 | 輝度範囲 | 成長補正 |
|------|---------|---------|
| 勇猛 | 最高 | 攻撃成長率↑ |
| 活発 | 高 | MP成長率↑ |
| 強靭 | 中 | HP成長率↑ |
| 堅固 | 低 | 防御成長率↑ |
| 神秘 | 最低 | 全ステ成長率小↑ |

### 1.4 生成ロジック（画像 → ペット）

| 解析ステップ | 手法 | 出力 |
|------------|------|------|
| 色解析 | 平均RGB | 属性（火/水/草/闇/光） |
| 輪郭解析 | エッジ検出 | 種類（5種） |
| 明るさ解析 | 輝度平均 | 性格（5種） |
| ノイズ解析 | 分散 | レア度 |

- 各ステップは独立関数として実装（差し替え可能）
- 同一画像 → 同一Petオブジェクト（再現性あり）
- 実装方式：Canvas API簡易版（ブラウザ完結）

### 1.5 ステータス上昇（餌やり）

- 通常：+1〜+5 ランダム上昇（性格補正を乗算）
- レア成長：低確率で +10
- 確率減衰：上限付近で段階的に減衰

| 現在値（上限比） | 上昇確率倍率 |
|----------------|------------|
| 0〜70% | ×1.0 |
| 71〜90% | ×0.5 |
| 91〜100% | ×0.1 |

- 全ステータス上限（各100）到達後：HP回復のみ発動（通貨消費あり）

### 1.6 レア度別成長ボーナス確率

| レア度 | レアボーナス発生確率 |
|--------|----------------|
| ★★★ レア | 20% |
| ★★ アンコモン | 10% |
| ★ コモン | 5% |

## 3. ゲームループ

```
画像アップロード
  → ペット生成
  → 餌やり（空腹度回復・HP+20・ステータス上昇）
  → 訓練（難易度選択・オートバトル）
  → 報酬獲得（EXP・通貨）
  → ユーザーレベルUP
  → 難易度上昇
```

---

## 4. 訓練（オートバトル）

- 進行：自動
- 難易度：3段階手動選択

| 難易度 | 係数 | 報酬倍率 |
|--------|------|---------|
| 易しい | ×0.5 | ×0.5 |
| 普通 | ×1.0 | ×1.0 |
| 難しい | ×1.5 | ×1.5 |

### 4.1 計算式

```
ペット総合力  = HP + MP + 攻撃 + 防御
難易度       = (ユーザーLv × 10 + ペット総合力 × 0.5) × 難易度係数
勝率         = ペット総合力 ÷ 難易度（0.1〜0.9クランプ）
EXP獲得量    = 難易度 × 0.5（勝利時のみ）
通貨報酬     = 8 × ユーザーLv × 報酬倍率（上限200）
```

### 4.2 結果処理

| 結果 | 処理 |
|------|------|
| 勝利 | EXP・通貨獲得・HP減少 |
| 敗北 | HP減少のみ・報酬なし |
| HP0 | 訓練不可・餌やりでHP+20回復後に再開可能 |
| 空腹度0 | 訓練不可・餌やりで回復後に再開可能 |

## 13. スキル

| 項目 | 値 |
|------|-----|
| 決定方法 | ペット生成時に性格と1対1対応で固定 |
| 発動タイミング | バトル開始時に確率判定 |
| 発動確率 | 30% |
| 発動条件 | MP > 0 |
| MP消費量 | 20 |
| MP=0時 | スキップ（訓練不可にはならない） |
| 効果適用 | 勝率補正（クランプ前加算 → 0.1〜0.9クランプ再適用） |
