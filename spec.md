# 仕様書 spec.md

## 1. ペット

### 1.1 データ構造

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string/number | 一意識別子 |
| typeIndex | number | 種類インデックス（encyclopediaFlags・アニメクラスの添字） |
| type | enum(5種) | 種類ラベル文字列 |
| level | number | ペットレベル |
| hp | number | 体力（上限100） |
| mp | number | スタミナ（上限100） |
| attack | number | 攻撃（上限100） |
| defense | number | 防御（上限100） |
| hunger | number | 空腹度 |
| personalityIndex | number | 性格インデックス（成長補正の参照に使用） |
| personality | string | 性格ラベル文字列 |
| attribute | string | 属性（火/水/草/闇/光） |
| rarity | string | レア度（将来拡張） |
| imageData | Blob | 元画像（IndexedDB保存） |

### 1.2 ペット種類（5種）

| 種類 | 輪郭特徴 | ゲームへの影響 |
|------|---------|--------------|
| ドラゴン系 | 縦横比縦長・エッジ密度中 | なし（図鑑のみ） |
| 鳥類系 | 重心上・縦横比横長 | なし（図鑑のみ） |
| 野獣系 | 重心中・エッジ密度高 | なし（図鑑のみ） |
| スライム系 | 縦横比正方形・エッジ密度低・重心下 | なし（図鑑のみ） |
| 精霊系 | エッジ密度最低・輪郭不明瞭 | なし（図鑑のみ） |

### 1.3 性格（5種）

| 性格 | 輝度範囲 | 成長補正 |
|------|---------|---------|
| 勇猛 | 最高 | 攻撃成長率↑ |
| 活発 | 高 | MP成長率↑ |
| 強靭 | 中 | HP成長率↑ |
| 堅固 | 低 | 防御成長率↑ |
| 神秘 | 最低 | 全ステ成長率小↑ |

### 1.4 生成ロジック（画像 → ペット）

| 解析ステップ | 手法 | 出力 |
|------------|------|------|
| 色解析 | 平均RGB | 属性（火/水/草/闇/光） |
| 輪郭解析 | エッジ検出 | 種類（5種） |
| 明るさ解析 | 輝度平均 | 性格（5種） |
| ノイズ解析 | 分散 | レア度 |

- 各ステップは独立関数として実装（差し替え可能）
- 同一画像 → 同一Petオブジェクト（再現性あり）
- 実装方式：Canvas API簡易版（ブラウザ完結）

### 1.5 ステータス上昇（餌やり）

- 通常：+1〜+5 ランダム上昇（性格補正を乗算）
- レア成長：低確率で +10
- 確率減衰：上限付近で段階的に減衰

| 現在値（上限比） | 上昇確率倍率 |
|----------------|------------|
| 0〜70% | ×1.0 |
| 71〜90% | ×0.5 |
| 91〜100% | ×0.1 |

- 全ステータス上限（各100）到達後：HP回復のみ発動（通貨消費あり）

| フィールド | 型 | 説明 |
|-----------|-----|------|
| level | number | ユーザーレベル（上限50） |

## 3. ゲームループ

```
画像アップロード
  → ペット生成
  → 餌やり（空腹度回復・HP+20・ステータス上昇）
  → 訓練（難易度選択・オートバトル）
  → 報酬獲得（EXP・通貨）
  → ユーザーレベルUP
  → 難易度上昇
```

---

## 4. 訓練（オートバトル）

- 進行：自動
- 難易度：3段階手動選択

| 難易度 | 係数 | 報酬倍率 |
|--------|------|---------|
| 易しい | ×0.5 | ×0.5 |
| 普通 | ×1.0 | ×1.0 |
| 難しい | ×1.5 | ×1.5 |

### 4.1 計算式

```
ペット総合力  = HP + MP + 攻撃 + 防御
難易度       = (ユーザーLv × 10 + ペット総合力 × 0.5) × 難易度係数
勝率         = ペット総合力 ÷ 難易度（0.1〜0.9クランプ）
EXP獲得量    = 難易度 × 0.5（勝利時のみ）
通貨報酬     = 8 × ユーザーLv × 報酬倍率（上限200）
```

### 4.2 結果処理

| 結果 | 処理 |
|------|------|
| 勝利 | EXP・通貨獲得・HP減少 |
| 敗北 | HP減少のみ・報酬なし |
| HP0 | 訓練不可・餌やりでHP+20回復後に再開可能 |
| 空腹度0 | 訓練不可・餌やりで回復後に再開可能 |

## 5. 経済設計

| 項目 | 値 |
|------|-----|
| 初期所持通貨 | 100 |
| 餌1個価格 | 10 × ユーザーLv |
| 餌の種類 | 1種類 |
| 訓練報酬 | 8 × ユーザーLv × 報酬倍率 |
| 訓練報酬上限 | 200（固定キャップ） |
| 餌やり効果 | 空腹度回復・HP+20・ステータス上昇 |

## 6. レベル設計

| 項目 | 値 |
|------|-----|
| 必要EXP | ユーザーLv × 100 |
| レベル上限 | 50 |
| Lv50後EXP | 0固定 |
| レベルアップ | 1回で1Lv（スキップなし） |

## 7. 画面一覧

| 画面 | 主な機能 |
|------|---------|
| ホーム（庭） | ペット表示・タップで下部パネルスライドイン |
| ケージ | 所持ペット一覧・庭に出すペット選択 |
| 生成 | 画像アップロード → ペット生成 |
| ショップ | 餌購入 |
| 訓練 | 3段階難易度選択・オートバトル |
| 図鑑 | 5種類の解放管理（未解放はシルエット表示） |

### 7.1 庭タップUI

- 表示形式：画面下部固定パネル（スライドイン）
- 表示内容：ステータス・空腹度・HP・種類・性格

### 7.2 庭ペット表示

| 項目 | 内容 |
|------|------|
| 素材 | 元画像を正方形クロップ・角丸処理（Canvas API） |
| アニメーション | 種類別CSSアニメーション |

| 種類 | アニメーション |
|------|--------------|
| ドラゴン系 | ゆっくり上下浮遊 |
| 鳥類系 | 左右揺れ |
| 野獣系 | 小刻み振動 |
| スライム系 | 上下つぶれ伸び |
| 精霊系 | フェードイン・アウト |
