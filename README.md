# たまてばこペット / Picto Garden

## 概要

画像ファイルからペットを生成し、育成・訓練するブラウザゲーム。
ジャンル：放置 × 育成 × 生成ガチャ

## タイトル

| 言語 | タイトル |
|------|---------|
| 日本語 | たまてばこペット |
| 英語 | Picto Garden |

## コアコンセプト

- 画像アップロード → Canvas API解析 → ペット生成（ブラウザ完結）
- 餌やり（ステータス上昇）＋ 訓練（オートバトル）のゲームループ
- ユーザーレベルと通貨による進行管理

## 世界観・デザイン方針

| 項目 | 方針 |
|------|------|
| 雰囲気 | ポップ・かわいい系 |
| カラー | ライト系（自然・草原） |
| 参考 | どうぶつの森系UI |

## 主要機能

| 機能 | 説明 |
|------|------|
| ペット生成 | 画像から属性・種類・性格・レア度を決定 |
| 庭 | ペットを表示・生活させる空間（初期1体・最大5体） |
| ケージ | 所持ペット管理・庭への配置選択 |
| 餌やり | 通貨消費 → 空腹度回復・HP+20・ステータス上昇 |
| 訓練 | 3段階難易度選択・オートバトル → EXP・通貨獲得 |
| 図鑑 | 10種類の解放管理（未解放はシルエット表示） |
| ショップ | 餌の購入（1種類・10×ユーザーLv） |

## ペット種類（10種）

ドラゴン系 / 鳥類系 / 野獣系 / スライム系 / 精霊系 / 水棲系 / 昆虫系 / 植物系 / 岩石系 / 幻影系

## ステータス

HP / MP / 攻撃 / 防御（上限は種類・性格・レア度による可変値）

## 技術スタック

ブラウザゲーム（HTML/CSS/JavaScript・Canvas API・IndexedDB）
