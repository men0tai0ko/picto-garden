### 時刻帯別背景色（庭のみ・CSSクラスで上書き）

| 時刻帯 | クラス | sky背景 | ground背景 |
|--------|--------|---------|-----------|
| 朝（6〜11時） | `.time-morning` | #F9C96A→#C8E6F5 | デフォルト維持 |
| 昼（12〜17時） | `.time-noon` | デフォルト維持 | デフォルト維持 |
| 夕（18〜21時） | `.time-evening` | #E8784A→#7A5080 | #8AB060→#2A5020 |
| 夜（22〜5時） | `.time-night` | #0A0E2A→#1A2550 | #3A5030→#1A2810 |

- スコープ：`#screen-garden` 配下のみ。他画面に影響しない
- 切替方式：`transition: background 3s ease` による滑らか遷移
- 夜の星：`#garden-sky::after` の `box-shadow` 疑似要素で静的配置

## 4. 画面別UIデザイン仕様

### 4.1 庭（ホーム）

| 要素 | 仕様 |
|------|------|
| 背景 | 空（グラデ）・草原（グラデ）の2層。時刻帯（朝/昼/夕/夜）に応じてCSSクラスで自動切替 |
| ペット表示 | 正方形クロップ・角丸16px・種類別CSSアニメーション |
| タップ | 画面下部パネルスライドイン |
| ステータスバー | Lv・所持コイン・EXPバー（上部常時表示・--color-main背景） |
| 庭フッター | screen-garden時に表示（デコボタン・アイテムトレイ開閉） |

#### ペットアニメーション（種類別）

| 種類 | CSSクラス | アニメーション内容 |
|------|----------|--------------------|
| ドラゴン系 | pet-anim-dragon | 上下浮遊（3.2s） |
| 鳥類系 | pet-anim-bird | 左右揺れ±6deg（2.4s） |
| 野獣系 | pet-anim-beast | 小刻み振動XY1px（0.18s） |
| スライム系 | pet-anim-slime | 上下つぶれ伸びscale（1.8s） |
| 精霊系 | pet-anim-spirit | フェードイン・アウト opacity0.35（2.6s） |
| 水棲系 | pet-anim-aqua | 緩やか揺れ±3deg+X4px（4.0s） |
| 昆虫系 | pet-anim-insect | 高速左右振動X2px（0.12s） |
| 植物系 | pet-anim-plant | ゆっくり上下浮遊（5.0s） |
| 岩石系 | pet-anim-golem | 低頻度微振動X1px（3.0s） |
| 幻影系 | pet-anim-phantom | フェード+右ドリフトX6px（3.5s） |

#### 庭アイテムトレイ

- 画面下部から上スライド（transform translateY）
- タブ切替（ペット / アイテム）
- 各カード：60×60px・選択時に--color-mpボーダー
- 配置モード中はデコボタンを--color-hp色に変化

#### 下部ステータスパネル表示内容

- 名前（16px太字）＋✏️編集ボタン＋🎲ランダムリネームボタン
- 種類・性格・属性・等級バッジ（各色）
- HP / MP / 攻撃 / 防御（バー＋statCaps対応）
- 満腹度（パーセント数値＋バー）
- 「餌」ボタン（現在価格表示）・「おみず」ボタン・「庭から外す」ボタン

---

### 4.2 ケージ

| 要素 | 仕様 |
|------|------|
| レイアウト | 2列グリッド |
| カード | ペット画像（72×72px）・名前 |
| 庭在中 | アクティブ枠（--color-mainボーダー） |
| 世代バッジ | generation≥1のペットは右下に「N世」バッジ（--color-accent） |

#### ケージフッター（screen-cage時）

| ボタン | 機能 |
|--------|------|
| ＋ 生成 | 生成画面へ遷移（screen-generate） |
| 🐣 繁殖 | 繁殖画面へ遷移（screen-breed） |
| ✏️ 編集 | 編集モード切替（野に放つボタン表示） |

---

### 4.3 繁殖画面（screen-breed）

| 要素 | 仕様 |
|------|------|
| 親選択 | ケージカードから2体選択 |
| フッター | 「繁殖する」ボタン・「キャンセル」ボタン（ケージ用3ボタンは非表示） |

---

### 4.4 訓練

| 要素 | 仕様 |
|------|------|
| ペット選択欄 | 名前11px＋種類9pxの2行・HP/満腹度警告インライン表示 |
| ステータスカード | 名前15px太字・種類/性格/属性/等級11px・2列グリッドステータス・給餌ボタン・おみずボタン・満腹ゲージ |
| 難易度選択 | やさしい / ふつう / むずかしい（3ボタン）・難易度選択ボタン直下に開始ボタン |
| ログ | スクロール可能テキスト（スクロール位置保持） |
| 結果 | overlay-cardでペット画像＋EXP・通貨・HP変動を表示 |

---

### 4.5 ショップ

| 要素 | 仕様 |
|------|------|
| タブ | 建物タブ / 植物タブ / 小物タブ（タブバー・--color-accentアクティブ色） |
| 商品グリッド | 2列グリッド（shop-item-card） |
| 商品カード | SVGアイコン（64×64px）・名前・価格（--color-accent）・購入ボタン |

---

### 4.6 図鑑

| 要素 | 仕様 |
|------|------|
| レイアウト | 5列グリッド（10種を2行で表示） |
| アイコンサイズ | 56×56px・border-radius 12px |
| 解放済み | 通常表示・種類名テキスト（太字） |
| 未解放 | シルエット表示（Canvas API黒塗り）・種類名淡色 |

---

### 4.7 生成画面

| 要素 | 仕様 |
|------|------|
| アップロードゾーン | 点線枠・タップで画像選択・プレビュー140×140px |
| 生成ボタン | 押下後Canvas API解析（連打防止） |
| 完了表示 | 種類・性格・属性・等級をresult-rowで表示 |
| フッター | 「キャンセル」ボタン（ケージ用3ボタンは非表示） |

## 5. 進化フレーム枠

| stage | CSSクラス | 見た目 |
|-------|----------|--------|
| 0 | なし | 枠なし |
| 1 | evo-stage-1 | シルバーbox-shadow 3px（--color-evo-stage1） |
| 2 | evo-stage-2 | ゴールドbox-shadow 3px（--color-accent）＋外側グロー8px |

- 適用箇所：庭・ケージ・訓練（選択欄・ステータスカード・結果overlay）
- 図鑑には適用しない

## 6. オーバーレイ一覧

| オーバーレイ | 表示タイミング | 内容 |
|-------------|--------------|------|
| ステータスパネル | 庭でペットタップ | 名前・種類/性格/属性/等級バッジ・HP〜防御バー・満腹度・給餌ボタン |
| バトル結果 | バトル終了後 | ペット画像・勝敗・EXP獲得・通貨増減・HP変動 |
| ペット生成完了 | 画像解析完了後 | 種類・性格・属性・等級表示 |
| レベルアップ演出 | EXP条件達成時 | レベル数・難易度上昇通知 |
| 庭スロット拡張 | Lv10/20/30/40到達時 | 拡張後スロット数 |
| 進化演出 | 餌やり後に総合力閾値超過時 | 進化メッセージ・フレーム枠変化 |
| 野に放つ確認 | ケージ編集モードで削除ボタン押下時 | ペット名・画像付き確認ダイアログ（overlay-cardスタイル） |

## 7. 画面遷移

```
起動
  → 庭（ホーム）← ボトムナビ全画面から到達可能
      ├── タップ → ステータスパネル（下部スライド）
      ├── ケージ → ペット選択 → 庭反映
      │     ├── ＋ → 生成画面（screen-generate）→ 生成完了 → ケージ
      │     ├── 繁殖 → 繁殖画面（screen-breed）→ 繁殖完了 → ケージ
      │     └── 編集 → 野に放つ確認ダイアログ
      ├── 訓練  → 難易度選択 → オートバトル → 結果overlay
      │     └── LvUP条件達成 → レベルアップ演出
      ├── ショップ → タブ切替（餌/アイテム）→ 購入
      └── 図鑑 → 10種類確認
```
