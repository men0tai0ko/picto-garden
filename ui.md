## 4. 画面別UIデザイン仕様

### 4.1 庭（ホーム）

| 要素 | 仕様 |
|------|------|
| 背景 | 空・草原・地面の3層 |
| ペット表示 | 正方形クロップ・角丸・種類別CSSアニメーション |
| タップ | 画面下部パネルスライドイン |
| ステータスバー | Lv・所持コイン・EXP（上部常時表示） |

#### ペットアニメーション（種類別）

| 種類 | アニメーション |
|------|--------------|
| ドラゴン系 | ゆっくり上下浮遊 |
| 鳥類系 | 左右揺れ |
| 野獣系 | 小刻み振動 |
| スライム系 | 上下つぶれ伸び |
| 精霊系 | フェードイン・アウト |

#### 下部パネル表示内容

- HP / MP / 攻撃 / 防御（バーで表示）
- 空腹度（パーセント数値＋バー）
- 種類・性格バッジ

### 4.2 ケージ

| 要素 | 仕様 |
|------|------|
| レイアウト | 2列グリッド |
| カード | ペット画像・名前・種類・性格 |
| 庭に出中 | アクティブ枠（緑ボーダー） |
| 空き枠 | 点線・＋アイコン → 生成画面へ |

### 4.3 訓練

| 要素 | 仕様 |
|------|------|
| 難易度選択 | やさしい / ふつう / むずかしい（3ボタン） |
| バトル画面 | ダーク系背景・自ペット vs 相手 |
| HPバー | 両ペットに表示 |
| ログ | スクロール可能テキスト |
| 結果 | EXP・通貨・HP変動をオーバーレイ表示 |

### 4.4 ショップ

| 要素 | 仕様 |
|------|------|
| ヘッダー | アクセントカラー・所持コイン表示 |
| 商品カード | アイコン・名前・説明・価格・購入ボタン |
| 価格表示 | コインアイコン＋数値 |

### 4.5 図鑑

| 要素 | 仕様 |
|------|------|
| レイアウト | 5種類を一覧表示 |
| 解放済み | 通常表示 |
| 未解放 | シルエット表示（Canvas API黒塗り） |

### 4.6 生成画面

| 要素 | 仕様 |
|------|------|
| アップロード | 画像選択ボタン・プレビュー表示 |
| 生成ボタン | 押下後にCanvas API解析 |
| 完了 | 種類・性格・属性をオーバーレイ表示 |

## 5. オーバーレイ一覧

| オーバーレイ | 表示タイミング | 内容 |
|-------------|--------------|------|
| ステータスパネル | 庭でペットタップ | HP・MP・ATK・DEF・空腹度・種類・性格 |
| バトル結果 | バトル終了後 | EXP獲得・通貨増減・HP変動 |
| ペット生成完了 | 画像解析完了後 | 種類・性格・属性表示 |
| レベルアップ演出 | EXP条件達成時 | レベル数・難易度上昇通知 |
| 庭スロット拡張 | Lv10/20/30/40到達時 | 拡張後スロット数 |

## 6. 画面遷移

```
起動
  → 庭（ホーム）← ボトムナビ全画面から到達可能
      ├── タップ → ステータスパネル（オーバーレイ）
      ├── ケージ → ペット選択 → 庭反映
      │     └── ＋ → 生成画面 → 生成完了オーバーレイ → ケージ
      ├── 訓練  → 難易度選択 → バトル → 結果オーバーレイ
      │     └── LvUP条件達成 → レベルアップ演出
      ├── ショップ → 餌購入
      └── 図鑑 → 種類確認
```
